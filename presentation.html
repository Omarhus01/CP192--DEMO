THE PROBLEM

Students can write recursion.
They struggle to explain why it works.

📖

TEXTBOOKS

Show you the syntax. Don't show you the call stack unwinding.

🧠

THE GAP

Understanding why each call waits, and how values travel back up the chain.

🎮

THE ANSWER

Make recursion visible. Let students watch it happen — not just read about it.

"You don't read about recursion. You watch it happen."

THE LEARNING PIPELINE

Three phases. One concept. Mastery by the end.

01 — GUIDED

Walkthrough

Dr. Omarito narrates each step. Clones animate live. Call stack updates in real time. Player just observes.

02 — SCAFFOLD

Fill the Gaps

Template provided. Player fills in the base case and recursive call. Hints available after 2 wrong attempts.

03 — FREE

Write It Yourself

Blank editor. Full function from scratch. This is where the learning actually sticks.

LEVEL 1 — THE FLOORS

def climb(floors):
  if floors <= 1: # base case
    return True
  return climb(floors - 1)

LEVEL 2 — THE COIN STACK

def count(coins):
  if coins == 0: # base case
    return 0
  return 1 + count(coins - 1)
